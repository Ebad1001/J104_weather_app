# 🌤️ Skyglass Weather

A stunning, responsive weather application built with a premium glassmorphism design. Get detailed, real-time weather data for any city around the world, fully localized to the city's timezone.

**[🚀 View Live Demo on GitHub Pages](https://Ebad1001.github.io/J104_weather_app/)**

## ✨ Features

- **Dynamic Glassmorphism UI**: A sleek, modern interface with beautiful blur effects and fluid animations.
- **Light & Dark Mode**: Seamlessly switch between themes. Your preference is automatically saved!
- **Interactive Backgrounds**: The background gently animates and shifts its color palette based on the current weather condition (clear, rainy, snowy, etc.) and active theme.
- **Comprehensive Weather Data**:
  - Current Temperature & "Feels Like"
  - Humidity & Cloud Cover
  - Wind Speed & Direction
  - UV Index with safety recommendations
  - Precipitation levels
- **7-Day Forecast**: Easily plan your week ahead with high/low temperature predictions.
- **Global Search**: Type in any city name to instantly fetch its weather profile.
- **No API Keys Required**: Powered by the reliable, free [Open-Meteo API](https://open-meteo.com/).

## 🛠️ Built With

- **HTML5** & **Vanilla CSS3** (No external frameworks, ensuring blazing-fast loads)
- **Vanilla JavaScript** (ES6+)
- **Open-Meteo APIs** (Weather Forecast & Geocoding)
- Google Fonts (DM Serif Display & DM Sans)

## 🚀 Running Locally

Since this app uses Vanilla JS and standard web technologies, there is no build process required.

1. Clone the repository:
   ```bash
   git clone https://github.com/Ebad1001/J104_weather_app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd J104_weather_app
   ```
3. Open `index.html` in your favorite web browser or use a local development server like Live Server (VS Code).
